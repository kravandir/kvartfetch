## Dependencies
- `bash`
- `ncurses-utils` (Termux, OpenSUSE)

## Installation
```zsh
git clone https://github.com/kravandir/kvartfetch
cd kvartfetch
chmod +x install
sudo ./install
#if you using termux run ./install -t
```
## Configuration
Edit your ``$HOME/.config/kvartfetch/config ``
